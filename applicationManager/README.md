# nodeExpressSqliteEJS
An Example Application Using Node + Expres + SQlite + EJS implementing a simple reference manager


## Using it

1) Clone the repo
2) Install the dependencies

```
npm install
```


3) Start the server

```
npm start
```

4) Point your browser to http://locahost:3000
